Zeiger und Strings (Übungen)

Übung (Ausgabe I): Was gibt dieses Programm aus? Überlege zuerst im Kopf/am Papier und probier es dann am Rechner aus.

char t[]="Wald";
char* p = t;
p++;
p++;
*p = *p + 2;
printf("%s\n", t);
printf("%s\n", p);

char str[]="abcdefghijk";

char* q = p + 4;
p=p+7;
while(p>q)

}

Übung (Ausgabe III): Was gibt dieses Programm (main + Funktion) aus? Überlege zuerst im Kopf/am Papier und probier es dann am Rechner aus.

void main()
{
    char str[]="abcdefghijk";
    printf("%c", getat(str, 5 ));
}
char getat(char* p, int n)
{
    char* q = p + n;
    return *q;
}
